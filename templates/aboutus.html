About US

SentiStock is an open-source sentiment analysis program , which analyzes social networking sites for optimal stock market investments. SentiStock is a python program which process tweets from twitter as input and classifies them using a Bayesian classifier.This helps us in identifying the potential stock. SentiStock does linear regression on stock values to predict the future stock values, thus enabling the user to find out how much he should invest in.
